About the Project

This project was created by Sana Amjad as part of the Reducetarian Fellowship to raise awareness and inspire engagement through interactive learning tools. As a multidisciplinary technologist passionate about ethical technology and public interest design, Sana conceptualized, designed, and implemented this initiative using principles of human-centered computing, data storytelling, and inclusive design.
